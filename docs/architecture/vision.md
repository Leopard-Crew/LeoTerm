# LeoTerm Vision

LeoTerm is a native Developer Console for Mac OS X 10.5.8 Leopard on PowerPC.

It is not a Windows Terminal port.
It is not a PowerShell clone.
It is not a generic cross-platform terminal emulator.

LeoTerm combines project-aware command execution, build logs, Finder interaction,
terminal workflows, and later LeooRexx capability access in the style of Cupertino 2009.

The guiding question is:

Would this feel like something Apple could plausibly have shipped for Leopard developers in 2009?

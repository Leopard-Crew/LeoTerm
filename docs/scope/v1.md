# LeoTerm V1 Scope

LeoTerm V1 is a native Leopard Developer Console.

Included:
- Project profiles
- Saved project actions
- Command execution through NSTask
- Build and test output
- Structured log view
- Finder reveal actions
- Basic drag and drop for project folders
- Property List based configuration

Explicitly not included:
- Full terminal emulation
- PTY support
- ncurses compatibility
- PowerShell-style object pipeline
- LeooRexx capability execution
- Windows Terminal behavior cloning
